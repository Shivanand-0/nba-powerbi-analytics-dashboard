## Overview
This project presents an interactive Power BI dashboard analyzing NBA games, team performance, and player statistics from the 2004 season to the most recent available data.

The dashboard enables users to explore season-wise scoring trends, team comparisons, player performance insights, and conference-level dynamics through intuitive visualizations and filters.

## Tools & Technologies
- Power BI
- DAX
- CSV datasets

## Key Features
- Season-wise scoring trend analysis
- Home vs Away performance comparison
- East vs West conference win percentage analysis
- Top player performance insights
- Interactive slicers for season, team, and player filtering
